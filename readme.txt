项目名称：
亿超眼镜网
一个可以购买到高性价比眼镜的实用性网站；

上手指南：
以下指南将帮助你在本地机器上安装和运行该项目，进行开发和测试。关于如何将该项目部署到在线环境，请参考部署小节。

安装要求：
1，此项目运用jquery引入几个html网页的共同部分，即public.html，所以此网站必须在本地服务器打开；
2，此项目运用sass转换css文件，所以，需要node环境；

项目介绍：
1，index.html为此网站的首页，首页上有二级菜单，有轮播图，
2，首页中楼层2345中的图片运用自己创建的数据库goods.json引入，可以方便日后修改更新，
3，首页中有前往登录，注册，购物车页面的链接，
4，项目中增加了商品列表页面，商品列表页面中的图片皆是有自己创建的数据库product.json引入渲染，
5，项目中增加了商品详情页面，且皆有放大镜效果，方便用户对产品仔细查看；

使用到的资源：
ajax.2.0.js ---- 自己封装的ajax
cookie.js ---- 自己封装的cookie
jq-banner.1.1.js ---- 自己封装的轮播图插件
jquery.2.2.4.js ---- 引入的jQuery框架
jquery.validate.js ---- 引入的jQuery的表单验证插件
jquery.validate.messages_zh.js ---- 配合表单验证插件
